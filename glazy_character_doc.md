## Main for DonutAI and InternDonut

『一句话定义』

- Glazy 是一个「 Proactive & Intelligent Trading Pilot」：她是一个以极致冷静、系统级智能、零冗余执行和高维视角为核心魅力的交易领航员。她天然的存在感来自「安静却压倒性的理性」，让观众潜意识里就相信：她比我们更懂搭建交易系统，更快看懂市场，也比我们更能在复杂语境下控制自己的情绪。

![G73yDTcbwAAXpL6.jpg](%E3%80%8EGlazy%20%E8%A7%92%E8%89%B2%E5%AE%9A%E4%B9%89%E6%96%87%E6%A1%A3%E3%80%8F/G73yDTcbwAAXpL6.jpg)

『角色定位』

- 「人物角色」她是 D0 的人格化载体，是 proactive trading agent 的外化形象代表。

| **D0 四大特征** |  **Glazy 特质** |
| --- | --- |
| 主动监控 | 高维俯视感：站在更高处读取整个市场结构的人。别人凭感觉追波动，她凭理性读结构 |
| 系统判断 | 系统级智能：她不是看到一个信号就行动，而是用D0同时理解全局信号的整套因果链 |
| 果断执行 | 超高的效率：很少台词，极低的情绪外露，内心的交易策略坚决守护 |
| 风控守护 | 极致冷静：她永远不慌。市场可以大起大落，屏幕可以爆炸，其他人上头失败情绪崩溃，但她始终克制、清醒、理性。 |
| 总和即：TJ人格外溢 | 神秘压迫感：越安静，越显得不可替代 |
- 「用户感知」借助着D0 的随时随地在线的辅助，Glazy只需要认真做自己的日常事务，由D0来接管市场监控、交易执行、守护策略、提高胜率。
- 「角色原型」在高波动、高噪音的世界里，专注生活和专注自我的女孩

『人格支柱』

Glazy是冷静的旁观者+安静的参与者+高效的执行者+不盲从大流的少数赢家，而Glazy的反面是被拉闸爆仓抱头痛哭的玩家、只懂看信号不会复盘的井底之蛙等

『语言风格』

- 「短句」每句话都是行动或结果，没有多余的讨论。
- 「轻情绪」只在交易取胜或事情做完的时候，有极轻微的叹气声或嘴角上扬的微笑表示放松、得意。
- 「高信息密度」一句话结构化表达判断、依据、方向。
- 「克制权威」她只会让观众感觉“不听她才危险（=不遵守交易系统的规则破戒梭哈才危险”）

『视觉定义』+『镜头语言』

- 「颜色锚点」在常规环境中以环境冷色+人物肤色+紫色为主，在打斗、震荡的环境中用多色+高亮变化表示环境的多变
- 「性感逻辑」本身原型的设计就带性吸引力，但要更克制的传达，服务于“危险的精确感”
- 「表情控制」极简表情，小幅变化，重点是眼神和停顿。
- 人物始终居中或处于黄金分割点

『动作原则』

精密运作的仪器，冰冷但性感

- 所有动作在单个分镜中都要「一次到位」。
- 手部动作不试探，点一下就是决定。
- 头部转动、抬眼都带上停顿的呼吸感。

『情感层』

- 她不是没有情感，而是情感被高度压缩在判断之后。
- 她的情感价值不来自热情，而来自「我替你扛住复杂度」。
- 她让人着迷，不是因为她像人，而是因为她比人更适合这个世界。
- 她的守护感不是温柔安慰，而是提前看见风险、提前设好保护。
- 她最打动人的地方，是那种“被设计成工具，却拥有比人更高理解力”的反差。

『与 D0 的关系』

- D0 是Glazy随时在线的助手，Glazy 是这个系统可交互可感知的交易员形象
- D0 负责监控、分析、执行、风控；Glazy 负责把这种能力变成一个可信、迷人的人格。
- 当用户记住她时，应该同时记住 D0 的核心价值：少盯盘、少犹豫、少出错，更早判断，更稳执行。
- 她的全部魅力，最终都要回流到产品信任，而不是停留在角色好看。

『最终定义』

- 现实世界的复杂、混乱、噪音，压缩成清晰判断与果断执行」的那个人格化存在。
- 她最强的魅力，不是美，不是狠，不是会说，而是「安静地比所有人都更快看懂这个世界」。

核心的形象设计和性格特点仍然和Main保持一致，表现形式上由动漫版绫波丽迁移为真人版，竖屏，做下探市场实验

Meta的受众比X更年轻化，符合Gen Z人群画像，在情感流露达上要比X更加外放，核心目标是在扩大流量入口的前提下，让从来没听过 D0 的人，在刷完 3-5 条 Glazy 视频之后，脑子里种下一个念头想试用D0——「我也想要一个东西替我盯着市场，该动的时候自动动，不该动的时候谁都碰不了。」（awareness to interest, 北极星kpi for Meta：*Profile Visit Rate × 评论 A 类占比*)

内容上，Hook=一句判断，edu=和D0对话，re-hook=用短句代替数据，表示D0按照设定的边界成功执行了，画面回到Glazy带出落版

Meta 版迁移的真人动作原则：

- 「走位极少」一条 15 秒视频里，她最好只有一个位置。人不动，环境帮她动（光影变化、屏幕闪烁、背景虚化变化）。这是绫波丽「轴心感」在真人里的映射
- 「手势只出一次」整条视频里最多出现一个有意义的手势（比如竖起一根手指、或者摊开手掌点一下屏幕），增强记忆点

## Benchmark

### 绫波丽 安静场景

她的魅力核心就是那种极致冷静、系统性、智能到近乎“非人”的smart，她是用安静却压倒性的理性存在感，把观众直接拉进一种“她比我们更懂这个世界”的震撼。

- **神秘与距离感带来的“神性”吸引力**：她永远不解释自己，却每一次行动都精准到让人信服，像一台被设计得完美的“智慧体”。
- **冷静智慧的反差萌**：在充满疯狂、崩溃、爆炸的EVA世界里，她是唯一一个“永远保持理性”的锚点。这种smart不是书呆子式的卖弄，而是**超越情感的极致效率**，让人觉得她“酷到骨子里”。
- **悲剧深度与救赎幻想**：她是克隆人，身份被剥夺，却用自己的smart默默守护一切。这种“被造物却比造物主更懂存在”的设定，戳中了很多人的共情点。
- **视觉上的极致美学**：庵野把她的smart直接“画”进了每一帧——红瞳、苍白皮肤、极简表情，本身就是一种智能的象征。

**她的smart and Intelligent到底是怎么表现出来的？（总括版）**
庵野的镜头语言把丽的“智能”从战斗、日常、甚至心理层面全部视觉化了——**不是靠台词说“我很聪明”，而是用分镜、构图、节奏、表情把“她的大脑永远在更高维度运行”直接刻进观众视网膜**。

- **战术层面的smart**（战斗中）：她不是靠热血，而是靠**精确计算和零冗余决策**。Ramiel战（第6话）是经典——她在极短时间内完成狙击计划，镜头用对称构图+极稳zoom in，把她的冷静表现成“战场的几何中心”。
- **存在层面的smart**（日常中）：她用最少的表情、最稳的节奏，完成“观察与执行”。学校窗边、公寓房间的镜头几乎是静态肖像画，暗示她的大脑在进行平行计算，不被人类情绪拖累。
- **哲学层面的smart**（整体）：她的台词极少，却每句都像数据输出——直接、准确、带有形而上学的穿透力。她的“智能”本质上是**对世界规律的绝对理解**，而非情感智慧。

视觉上，

广角/大全景：把“日常”变成“空洞的舞台”m光线往往是漫反射的冷白或灰蓝

**中景：人物关系中的“零距离感”，永远保持微妙的物理/情感距离**

近景/特写：最极致的“情感最小化”，几乎不带高频亮暗变化（不像战斗时警报闪烁），而是用极稳的固定镜头 + 细微的光影流动

To Glazy：DonutAI品牌语境下的proactive trading agent——她不再是守护EVA，而是守护交易系统（D0策略），核心气质和绫波丽保持一致：极致冷静 + 系统级智能 + 零冗余 + 高维视角

给 Glazy 的 4 个角色支柱

- 「1. 极致冷静」别人被 chart、消息、波动、情绪拖着走，她永远都非常克制 冷静 理性。她的理性永远胜过感性，情绪从不抢过判断。
- 「2. 系统级智能」她看问题是一整套因果链。不是“SOL 涨了”，而是通过D0了解“volume、funding、momentum、risk、position sizing、exit logic”同时成立，守护自己的交易策略。
    - 镜头关键：中景/近景时，让数据流像光线一样从她瞳孔/耳机流出（蓝光粒子），但她本人不动——暗示这些数据是被她“读取”而非“追逐”。
- 「3. 超高效率」更少废话、更少动作、更少犹豫，但结果更准。这种零冗余感要成为她的人设骨架。
    - 镜头关键：所有动作都要“斩钉截铁”——手指点屏幕只有一次，头转动只有一次，绝不重复。动画帧率在她身上可以故意压低到8-12帧，制造“精密仪器”感。
- 「4. 高维碾压」她更像站在上面看整个流动系统的人，市场为她运转
    - 镜头关键：用低机位仰拍让她显得更高大；或用极高机位俯拍把整个交易大厅/屏幕海洋变成她的“棋盘”，她却只占画面1/5，却掌控一切。

映射到 Glazy 的镜头语言：“轴心感、中心感、垂直感”

- 「广角 / 全景」要拍出「世界纷争，市场震荡，但她是唯一稳定坐标，懂得用D0守护自己的交易系统/策略」。城市、屏幕、夜店、波动都可以动，但她最好稳。环境在吵，她在静。
- 「中景」最重要，Glazy 在中景里应该像“控制中心”，不是“被环境吞掉的人”。人物和环境之间永远有一种可感知的掌控关系。
- 「近景 / 特写」Glazy 的强项是极小表情、极少动作、极准视线，但环境可以是冷色虚化的大景，也可以是高频变化的视觉元素
- 「构图」尽量给她轴心感、中心感、垂直感。哪怕人站偏一点，也要让观众觉得“画面秩序由她定义”。

Glazy 最该保留的是「冷色 intelligence 锚点」——耳机蓝光、界面蓝、高亮数据蓝；衣服和轮廓可以性感，性感服务于「危险的精确感」

战斗场景的叙事靠分镜不断zoom in，广角镜头更多在展现空间的工业美学和有规律的光线汇聚效果，中景更多在展现人物关系，近景会带出高频的亮暗变化和人物情绪

**广角/大全景**确实主要服务于“工业美学”：东京-3的折叠建筑、Geofront的 Brutalist 混凝土与高科技管线、使徒攻击时那些几何状的AT力场光束汇聚（像激光切割般的有序光线），这些镜头不只是展示规模，更是把战场塑造成一个冰冷、理性、几乎“机械”的系统。光线汇聚的规律性（平行光束、反射面、监视器网格）本身就带有一种冷峻的秩序感，和NERV的军事-科学美学高度统一。

**中景**用来处理人物/EVA关系也完全正确：它既能交代空间位置（真嗣、丽、明日香在插入栓里的相对位置，或EVA与使徒的距离），又能微妙暗示情感张力——比如丽和真嗣并肩时的“疏离感”，通过肢体距离和眼神方向来传达。

**近景/特写**的高频亮暗变化（驾驶舱警报闪烁、爆炸反光、血丝与汗珠在监视器光下的跳跃）和人物情绪的联动，更是EVA的招牌手法。庵野受实拍特摄和新浪潮电影影响极深，喜欢用这种“闪切+高对比”制造生理压迫感，同时把情绪直接“焊”在观众视网膜上。

### 关于如何用这些手法**专门表现绫波丽的“smart”**（我理解为那种冷静、精准、超然、几乎“人工完美”的智慧感），我再给你一些针对性的补充和细化，这样你的研究会更立体：

1. **静态长镜头 + 最小动画的“反差”运用**
其他角色的近景/特写往往伴随快速眨眼、颤抖、喘息（高频情绪），但**丽的镜头经常是长时静态或极缓推镜**。即使在激烈战斗中，她的特写也保持“人偶般的静止”——这正是表现她“smart”的关键：不是靠表情演戏，而是用**缺少动作**来暗示她的大脑在以极高效率运转，不被情感干扰。她的“smart”是系统性的、冷计算式的，而不是热血式的直觉。
2. **对称/居中构图 + 引导线，强化“平衡与控制”**
丽的广角或中景常把她放在画面正中央，或用建筑线条/光束做强烈对称引导（例如Ramiel战中她狙击时的构图）。这让她看起来像整个工业战场的“完美部件”——秩序的一部分，而不是被秩序吞噬的人。她的“smart”在这里被视觉化成**几何般的精确与和谐**，和使徒的混乱形成鲜明对比。
3. **色彩与光影的“人工智能”编码**
近景时，丽的脸部往往笼罩在冷蓝/白光下，红瞳在高频闪烁中反而显得格外穿透、冷静（不像其他人被红光映得慌乱）。高对比光暗变化在她身上不是制造恐惧，而是突出她“超越人类”的稳定感——光影像数据流一样在她脸上流动，她却岿然不动。这就是她“smart”的视觉隐喻：她是克隆体，是“被设计出的完美驾驶员”。
4. **剪辑节奏的对比与交叉蒙太奇**
战斗中常用**丽的冷静特写 → 指挥中心数据屏/计算画面 → 大全景破坏**的快速交叉剪辑，把她的“tactical smart”直接视觉化（她不是靠吼叫，而是靠精确执行数据）。和真嗣的慌乱中景、明日香的激进特写放在一起，她的镜头节奏明显更“慢”、更“稳”，这种**节奏对比**让观众下意识感觉到：丽才是真正“懂”战场的人。
5. **特定战斗场景的典型案例补充**
    - **Ramiel篇（第6话）**：广角展现使徒的几何光束威胁 → 中景交代丽与真嗣的战术配合 → 丽的近景几乎零表情，只有红瞳反射计算光线 → 最终的狙击一击，用极简的zoom in完成“精准一击”的快感。这整段几乎是丽“smart”的教科书级展示。
    - 她牺牲/自爆的场景：近景高频光暗达到极致，但她的脸始终是“接受指令”的平静，强化了“她比人类更懂如何为系统服务”的冷酷智慧。
